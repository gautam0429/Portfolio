# Portfolio Website - Gautam Agrahari 🚀

Full Stack MERN Developer portfolio built with React, Three.js, and GSAP.

---

## 🛠️ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/gautam0429/Portfolio-Website.git
   cd Portfolio-Website
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```
   *Note: If you encounter issues with GSAP Trial plugins, ensure you have an active internet connection or refer to GSAP documentation.*

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
